Sage

The HTML format of the textbook takes advantage of the Sage programming language. To become acquainted with Sage, see how this textbook uses Sage. Instructors can either assign this section to the students, or spend some time in class to illustrate how Sage works.

Each section of the HTML format of textbook contains Sage exercises. These exercises vary in complexity. The goal of the exercises is to get users to use technology to explore structures and make conjectures that would be very difficult to explore by hand computation. For example, the Sage Exercises in Cyclic Groups were written to help readers explore the cyclic patterns of the group of units mod \(n\text{,}\) \(U(n)\text{,}\) which is sometimes cyclic, sometimes not. By using Sage, users can easily test whether or not the \(U(n)\) is cyclic for large values of \(n\) and then make a conjecture about what the general case should be.
