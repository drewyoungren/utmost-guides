Sage is an open source language for doing mathematics. It is designed to replicate the functionality of popular computer algebra systems (Mathematica, MAPLE, etc.) in a unified, open source environment. It is free and may be downloaded and run locally on a user's personal computer. Many PreTeXt authors deploy what are known as Sage Cells like those seen below. These are one-off computing environments in which snippets of code can be executed and displayed in a browser without need for any installation of software on the user's part. The code can be altered in any way the user sees fit, and the original content can be restored by reloading the page.
