Sage

The textbook makes heavy use of Sage cells (there are 96 vignettes with Sage cells, a list of which can be found in the Sage appendix 1 ). These are intended to help students learn to use Sage to do linear algebra and to learn how to do linear algebra by using computational examples. There is a separate GitHub repository that contains 17 Sage worksheets that instructors can use in class (in several output formats, including Jupyter, PDF, etc.). They are "incomplete" and are meant to be filled in during class. For some examples, random elements are generated; in others, a carefully constructed example is explicitly defined. There is a master document with a short description and the expected duration (usually between 5 and 20 minutes) for each. Instructors can do them in class; they can see many different examples in class.
